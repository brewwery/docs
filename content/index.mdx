# Brewwery

**A native macOS GUI for Homebrew.**

Brewwery puts packages, casks, services, updates, and diagnostics in one clean interface. Open source, local-first, zero config — just launch and manage your entire Homebrew setup visually.

## What is Brewwery?

Brewwery is a macOS desktop app that gives you a visual interface for everything Homebrew manages. Instead of memorizing terminal commands, you get a clean dark UI with sidebar navigation, package lists, one-click updates, service controls, and diagnostic tools.

## Key Highlights

- **Visual package management** — see all installed formulae and casks in a filterable, sortable list
- **One-click updates** — see outdated packages and upgrade them individually or all at once
- **Service control** — start, stop, and restart Homebrew services without terminal commands
- **Cleanup & diagnostics** — preview disk savings before cleanup, run `brew doctor` visually
- **Brewfile export** — export reproducible Brewfiles and track local install history
- **Fast Rust backend** — commands are parsed and validated in Rust, no shell spawning
- **Local-first** — no telemetry, no cloud sync, no auth, no paid logic
- **Open source** — MIT licensed, developed in public

## Current Status

Brewwery is at **v0.9.1 — Discover & Favorites**. The latest release adds a Favorites system so you can mark and quickly access your most-used packages, alongside Search & Discovery improvements.
